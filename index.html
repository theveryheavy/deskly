Why Deskly

Everything your support team needs

Built for multi-server scale from day one. No config files, no JSON editing — everything lives in PostgreSQL.

Button-based ticketing

Ticket panels with one-click open buttons. No commands needed for users — just click and a private channel appears instantly.

Private channel system

Every ticket gets its own Discord channel with proper permission overwrites. Only the opener and support staff can see it.

Ticket claiming

Staff can claim tickets to signal ownership. Prevents duplicate replies and builds clear accountability across your team.

Automatic transcripts

When a ticket closes, a full plain-text transcript is saved to PostgreSQL and posted to your log channel as an attachment.

Add and remove users

Staff can add or remove members from any ticket mid-conversation. Perfect for escalations and department handoffs.

Full multi-server isolation

Each Discord server has its own config, roles, and ticket counter stored in PostgreSQL. Nothing bleeds between guilds.

Getting started

Up and running in three steps

No config files, no JSON editing, no CLI wizards. Everything is a Discord command.

1

Invite the bot

Click Add to Discord and grant the required permissions. Deskly registers your server automatically on join.

2

Run /setup

Configure your support role, admin role, ticket category, and log channel. All stored in the database — no files touched.

3

Post with /panel

Drop a ticket panel in any channel. Your users see a button, click it, and a private support channel appears. Done.

Slash commands

Simple to set up, easy to use

Six commands cover everything — setup, management, and information.

/
setup
Configure roles, ticket category, and log channel for your server.
/
panel
Post a ticket panel embed with an Open Ticket button in any channel.
/
reset-config
Wipe all bot configuration for your server with a confirmation step.
/
ping
Check bot roundtrip latency and WebSocket heartbeat in real time.
/
help
Show all available commands with descriptions in a clean embed.
/
vote
Show voting links for all bot listing platforms with one-click buttons.
